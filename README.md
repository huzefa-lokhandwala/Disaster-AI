# DisasterPrep – Disaster Management Training Platform

## Features
- Virtual Disaster Drill Simulation
- AI Chatbot for Disaster Help
- Preparedness Score System
- Leaderboard
- Admin Dashboard
- Emergency Alert System
- Disaster Learning Modules

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: SQLite
- Charts: Chart.js
- Maps: Leaflet.js
- AI: OpenAI API

## How to Run
pip install -r requirements.txt
python app.py
